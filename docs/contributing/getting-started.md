## Prerequisites

- Node.js 20+
- pnpm
- A local Postgres database, such as Docker Desktop or a native install, or a Supabase Postgres project

## Quick start

Follow the [Quick Start](/doc/development/quick-start) page for the contributor flow. It covers forking, cloning, local Postgres setup, the Supabase storage env vars for image and upload features, and the local development commands in order.
